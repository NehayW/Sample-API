# Sample-API

## About

Sample Api calling app with the help of React redux saga

## Installation

### For Android

        - npm install/yarn install
        - cd android && ./gradlew clean
        - cd ..
        - npx react-native run-android / yarn android
        - npm start / yarn start

### For IOS

        - npm install/yarn install
        - cd ios && pod install
        - cd ..
        - npx react-native run-ios / yarn ios
        - npm start / yarn start
